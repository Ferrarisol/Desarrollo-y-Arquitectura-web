      HTML5 y sus elementos avanzados

      HTML5 es una versión moderna del lenguaje de marcado que permite estructurar páginas web de forma clara y organizada.

      Etiquetas HTML más complejas

      HTML5 incorpora etiquetas semánticas como header, nav, section, article y footer,que ayudan a organizar el contenido de manera más eficiente.

      Elementos de bloque y de línea

      Los elementos de bloque ocupan todo el ancho disponible y generan un salto de línea,como los párrafos o divisiones.

      Los elementos de línea ocupan solo el espacio necesario y no generan salto,omo los enlaces o imágenes.

      Inputs, botones y formularios

      Los formularios permiten ingresar datos en una página web. Se utilizan inputspara capturar información y botones para enviar los datos.

      HTML5 permite validar automáticamente los datos ingresados por el usuario.
